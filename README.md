# med

## requirements

- docker cli
- aws sam cli
- aws cdk cli
- make sure docker is running
  - `sudo service docker start`
- run `npm install`
- run `cd cdk && npm install`

## development

- `sh watch-lambda.sh`
- `sh watch-server.sh`
- `sh watch-client.sh`
